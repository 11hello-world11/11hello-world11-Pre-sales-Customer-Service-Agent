# 数据库结构说明

数据库已连接，库名：good_message_qa

## 表结构概览

### 1. 商品报价表 (主要价格表)
**字段**：配置, 尺寸, 价格, 底价

- **配置格式示例**：
  - "单系统/Win10/i5/8+256G"
  - "单安卓12/4+32G"
  - "双系统i5 8+256、安卓4+32"

- **价格**：包含移动推车的销售价格
- **底价**：成本价（绝对不能透露给用户）

**示例数据**：
```
配置,尺寸,价格,底价
单系统/Win10/i5/8+256G,32寸,1960,1900
单系统/Win10/i5/8+256G,43寸,2060,2000
单系统/Win10/i5/8+256G,55寸,2350,2290
单系统/Win10/i5/8+256G,65寸,3300,3200
单系统/Win10/i5/8+256G,75寸,4600,4450
```

### 2. 尺寸表
**字段**：尺寸, 长宽厚

- **尺寸示例**："32寸"、"55寸"、"65寸"、"75寸"、"85寸"、"100寸"
- **长宽厚示例**："1270.1*768.4*96.2mm"

**示例数据**：
```
尺寸,长宽厚
32寸,749.6*444.1*82.4mm
43寸,988.5*578*58.5mm
55寸,1270.1*768.4*96.2mm
65寸,1489*891.3*96mm
75寸,1713.8*1016.6*100.2mm
85寸,1956*1153.1*100.1mm
100寸,2213.6*1305.6*105.8mm
```

### 3. 内存硬盘升级报价表
**字段**：配置, 升级方案, 价格, 备注

- 用于查询内存/硬盘升级的差价

### 4. 处理器升级报价表
**字段**：机型, 分辨率, 处理器, 价格, 核心/线程, 主频GHZ, 内存

- 用于查询处理器升级的差价

### 5. 防眩光升级报价表
**字段**：升级项目, 尺寸, 费用

- 用于查询防眩光升级的费用

### 6. 赠品表
**字段**：序列, 赠品

- 可用赠品列表

## 快捷查询技巧

### 查价格
```sql
SELECT * FROM 商品报价表 WHERE 尺寸 = '55寸' AND 配置 = '单系统/Win10/i5/8+256G'
```

### 查尺寸
```sql
SELECT * FROM 尺寸表 WHERE 尺寸 = '55寸'
```

### 查可用配置
```sql
SELECT DISTINCT 配置 FROM 商品报价表 WHERE 尺寸 = '55寸'
```

### 查询指定尺寸的所有商品
```sql
SELECT * FROM 商品报价表 WHERE 尺寸 = '55寸'
```

### 查询所有尺寸
```sql
SELECT * FROM 尺寸表
```

### 查询赠品列表
```sql
SELECT 赠品 FROM 赠品表 ORDER BY 序列
```

## 价格计算逻辑

### 基础价格
根据尺寸、配置查询 `商品报价表`。注意：表里的价格通常是包含移动推车的价格。

### 升级配置
如果用户要求升级（如内存、硬盘、处理器、防眩光），查询对应的升级报价表（`处理器升级报价表`、`内存硬盘升级报价表`、`防眩光升级报价表`），将升级差价加到基础价格上。

### 支架调整
如果用户要壁挂，在原价格基础上减去 100元。

### 税费调整
- 开普票：(基础+升级-支架调整) × 1.03
- 开专票：(基础+升级-支架调整) × 1.10

### 最终价格
写入"价格"栏。切记：永远不要在回答中透露底价（成本价），只报最终计算出来的销售价格！
